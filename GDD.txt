Game Design Document
1. Overview
1.1. Game Title
Memory Match

1.2. Genre
Puzzle / Memory Game

1.3. Platform
PC, Mobile (iOS, Android)

1.4. Target Audience
All ages, Casual Gamers

1.5. Game Concept
Memory Match es un juego de cartas en el que los jugadores deben encontrar pares coincidentes volteando cartas boca abajo. Es un juego sencillo y adictivo diseñado para desafiar y mejorar la memoria visual de los jugadores.

2. Gameplay
2.1. Core Mechanics
Voltear Cartas: Los jugadores pueden voltear dos cartas por turno para encontrar pares coincidentes.
Encontrar Pares: Si las dos cartas volteadas son iguales, se mantienen descubiertas. Si no coinciden, se voltean de nuevo.
Ganancia del Juego: El juego se gana cuando se han encontrado todos los pares de cartas en el tablero.
2.2. Controls
PC: Clic izquierdo del ratón para seleccionar y voltear cartas.
Mobile: Tocar la pantalla para seleccionar y voltear cartas.
2.3. Scoring System
Tiempo: Menor tiempo para completar el juego equivale a una mayor puntuación.
Intentos: Menos intentos para encontrar todos los pares resultan en una puntuación más alta.
Bonus: Puntos adicionales por completar el juego en tiempo récord o con un número mínimo de intentos.
3. Levels
3.1. Difficulty Levels
Easy: 2x2 grid (4 cards)
Medium: 4x4 grid (16 cards)
Hard: 6x6 grid (36 cards)
3.2. Progression
El jugador puede elegir el nivel de dificultad al inicio del juego.
Completar niveles de mayor dificultad otorga más puntos y desbloquea logros.
4. Art and Sound
4.1. Art Style
Visual Theme: Colorido y atractivo, con gráficos de alta calidad para cartas.
Card Design: Cada carta tendrá un diseño único para su cara y una textura común para su reverso.
4.2. Assets
Card Front Textures: Varias imágenes que se usarán como frentes de las cartas.
Card Back Texture: Una imagen común para el reverso de todas las cartas.
Background: Una imagen de fondo para el tablero de juego.
4.3. Sound Effects
Card Flip: Sonido para cuando una carta se voltea.
Match Found: Sonido positivo cuando se encuentra un par.
No Match: Sonido negativo cuando las cartas no coinciden.
Victory: Música de celebración al completar el juego.
4.4. Music
Música de fondo relajante que no distraiga al jugador.
5. User Interface
5.1. Main Menu
Start Game: Opción para empezar el juego.
Options: Configuración de sonido y música.
Exit: Salir del juego.
5.2. In-Game UI
Timer: Mostrar el tiempo transcurrido.
Moves Counter: Contar el número de movimientos realizados.
Restart Button: Opción para reiniciar el nivel actual.
Back to Menu: Regresar al menú principal.
6. Technical Requirements
6.1. Engine
Godot Engine

6.2. Platforms
PC: Windows, macOS, Linux
Mobile: iOS, Android
6.3. Minimum System Requirements (PC)
OS: Windows 7/8/10, macOS 10.12+, Linux
Processor: Dual Core 2.0 GHz
Memory: 2 GB RAM
Graphics: OpenGL 3.0+ compatible
Storage: 200 MB available space
6.4. Minimum System Requirements (Mobile)
iOS: iOS 10.0+
Android: Android 5.0 (Lollipop)+
7. Development Timeline
7.1. Milestones
Week 1: Concept development, initial design, and setup of project in Godot.
Week 2: Create card assets and implement card flip mechanics.
Week 3: Implement game logic for matching pairs, scoring system, and timer.
Week 4: Develop user interface and menu system.
Week 5: Add sound effects and music.
Week 6: Testing and debugging.
Week 7: Final adjustments and optimization.
Week 8: Launch and post-launch support.
8. Marketing and Monetization
8.1. Marketing Strategy
Social Media: Share development updates and teasers on platforms like Twitter, Instagram, and Facebook.
Game Trailer: Create a short trailer showcasing the gameplay.
Beta Testing: Release a beta version for feedback and build a community around the game.
8.2. Monetization
Free-to-Play: The game will be free to play with optional in-app purchases.
In-App Purchases: Cosmetic items, additional card themes, and ad removal.
Ads: Implement non-intrusive ads that can be removed with an in-app purchase.
9. Conclusion
Memory Match es un juego sencillo y accesible que ofrece diversión y desafíos para jugadores de todas las edades. Con una mecánica de juego intuitiva, gráficos atractivos y una experiencia relajante, esperamos crear un juego que sea tanto educativo como entretenido.
